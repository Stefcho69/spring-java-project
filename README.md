# spring-java-project
This is my website i made in Java as my Softuni project. It's main purpose is to act as a online shop. 

# It uses:
Java version: 17                  
Spring Boot version: 3.4.0                  
Maven as a Build Tool                 
MySQL for relational db                      
Github for source control

The main application is made with MVC and Thymeleaf                              
The supported microservice uses REST API                        
They are connected by Feign                              

# Commit types used in this repo:
feat - new feature                  
fix - bug fix                      
refactor - code cleanup/refactor                   
test - tests added/updated                      
docs - documentation changes                            
chore - minor updates (e.g., config)
